# PanicTab 隐私政策

生效日期：2026-07-01

PanicTab 是一款 Chrome 扩展，用于在用户主动触发时快速整理当前窗口标签页，并打开可自定义的遮挡页面。

## 我们收集的数据

PanicTab 不会收集、出售、共享或远程传输用户数据。

扩展会在浏览器本地保存以下设置：

- 鼠标三连击触发是否启用
- 网页内键盘触发是否启用
- 网页内键盘触发方式
- 标签组名称
- 标签组颜色
- 遮挡页面类型
- 界面语言偏好
- 用户填写的远程图片 URL
- 用户填写的伪装网页地址
- 用户上传的本地图片数据

这些设置仅保存在用户本机浏览器的 `chrome.storage.local` 中。

## 权限用途

PanicTab 使用 `tabs` 权限读取当前窗口中的标签页，以便在用户主动触发时将标签页归入同一个标签组。

PanicTab 使用 `tabGroups` 权限创建和更新 Chrome 标签组，包括设置标签组名称、颜色和折叠状态。

PanicTab 使用 `storage` 权限在浏览器本地保存用户设置。

PanicTab 使用 `http://*/*` 和 `https://*/*` 主机权限在普通网页中运行内容脚本，用于监听用户主动配置的鼠标三连击和网页内键盘触发。内容脚本不读取网页正文，不分析页面内容，也不会上传页面数据。

## 数据共享

PanicTab 不会将用户数据传输给开发者或任何第三方。

如果用户主动填写远程图片 URL，浏览器会向该 URL 所属服务器请求图片。如果用户主动填写伪装网页地址并选择该模式，浏览器会直接访问该网页。上述请求不由 PanicTab 服务器处理，因为 PanicTab 不提供远程服务器。

## 广告与追踪

PanicTab 不包含广告，不进行个性化广告投放，不进行跨站追踪。

## Limited Use 声明

PanicTab 对信息的使用遵守 Chrome Web Store User Data Policy，包括 Limited Use 要求。PanicTab 仅将权限和本地数据用于提供或改进扩展的单一用途：在用户主动触发时整理当前窗口标签页并打开用户配置的遮挡页面。
