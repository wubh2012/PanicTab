# PanicTab

PanicTab 是一个轻量级 Chrome Manifest V3 扩展，可以快速整理当前窗口中打开的标签页，将它们归入一个折叠的标签组，并打开预设遮挡页面。适合屏幕共享前整理浏览器、公共空间保护屏幕隐私，或在有人靠近屏幕时快速切换到安全页面。

## 功能

- 在普通 `http` 和 `https` 网页中默认支持双击 `Ctrl` 触发。
- 可在设置页手动启用鼠标左键三连击触发，默认关闭。
- 支持点击扩展工具栏图标触发。
- 支持 Chrome 原生快捷键触发，默认是 `Ctrl+Shift+Y`，macOS 默认是 `Command+Shift+Y`。
- 非破坏式整理标签页：只分组并折叠，不关闭标签页。
- 设置页支持启用触发方式、选择网页内键盘触发、设置分组名称和颜色，并可选择浏览器崩溃页、自定义图片或用户设置的网页地址作为遮挡页面。
- 扩展界面支持简体中文、英文、日文、韩文、德文和法文，可在设置页切换；商店上架材料与隐私政策目前提供中英双语。

## 适用场景

- 有人靠近屏幕时，一键收起标签页并打开预设遮挡页面。
- 会议共享或现场演示前，快速整理杂乱浏览器窗口。
- 咖啡馆、图书馆、开放办公区等公共空间中，减少屏幕内容被旁人看到的风险。

## 在 Chrome 中加载

1. 打开 `chrome://extensions`。
2. 启用开发者模式。
3. 点击 `加载已解压的扩展程序`。
4. 选择本项目目录。

如需修改 Chrome 原生快捷键，打开 `chrome://extensions/shortcuts` 并编辑 PanicTab 命令。

## 打包

在项目根目录执行：

```powershell
.\scripts\package.ps1
```

脚本会生成 `dist/PanicTab` 和 `dist/PanicTab-v<版本号>.zip`。上传 Chrome Web Store 时使用 zip 文件即可。

## 说明

Chrome 真实的内部崩溃页或错误页不能由扩展可靠打开或控制。PanicTab 使用本地遮挡页面，以保证行为稳定且可配置。

用户选择“网页地址”伪装方式时，PanicTab 会直接在当前窗口打开用户保存的 `http://` 或 `https://` 地址；地址为空或无效时会回到内置遮挡页面。
